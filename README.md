# my-class12th-practical-project
Secure Bank Management System A Python-MySQL banking app with OTP verification, fund transfers, fixed deposits, loans, and admin controls. Features dual user/admin panels, secure login with email OTP, transaction limits, KYC verification, and real-time email alerts. Demonstrates database connectivity, encryption, and secure transaction handling.
Banking Application Project - Class 12th Practical
Project Overview
Project Name: Secure Bank Management System

Developers: Rupesh, Jayaraju, Pranathi, Pooja

Brief Description
This is a Python-based Banking Application that simulates a real-world banking system with dual interfaces for Users and Administrators.

Key Features:
For Users:

Register with KYC verification (Aadhaar, PAN, etc.)

Secure login with OTP on email

Deposit, withdraw, and transfer money

Add/view beneficiaries for quick transfers

Apply for Fixed Deposits and Loans

View transaction history

Update profile and change password

For Admins:

Manage all user accounts

Approve/reject loan applications

View system statistics and audit logs

Configure system settings

Technologies Used:
Python (Core programming)

MySQL (Database for storing user data)

SMTP (For sending OTPs and email alerts)

Hashlib (For password encryption)

Database Tables:
Users, Transactions, Beneficiaries, Fixed Deposits, Loans, Admins, Audit Logs, etc.

This project demonstrates real-world banking operations with proper security measures like OTP verification, password hashing, transaction limits, and audit logging.
